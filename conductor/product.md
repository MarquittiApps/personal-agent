# Product Context (product.md)

**Product:** Personal AI Core
**Vision:** Centralizar a gestão profissional e pessoal num único fluxo "Agentic", acessível via Web e IDE.

## Main Modules
- **Supervisor (Meta-Agente):** Orquestração e auto-evolução.
- **Daily Planner:** Gestão pessoal e tempo.
- **Software Dev (ClinicCare):** Engenharia de software sênior.
- **Smart Home:** Projetos IoT e automação.
- **Consultoria TI:** Backoffice e infraestrutura física.

## Core Goals
- Reduzir carga administrativa em 80%.
- Acelerar desenvolvimento do ClinicCare.
- Padronizar projetos de engenharia.
- Prover Interface Web premium (Human-in-the-Loop).
