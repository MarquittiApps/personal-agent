# Specification: MVP Planejamento Diário com IA

**Track ID:** mvp-planejamento_20260318
**Type:** Feature
**Created:** 2026-03-18
**Status:** Draft

## Summary

Permitir que o usuário consulte e gerencie seus compromissos e tarefas (Google Calendar e Tasks) via chat conversacional, resultando em uma seção "Planejamento" consolidada com insights gerados por IA para otimizar o dia.

## Context

Este módulo atua como Assistente Pessoal (Daily Planner), estabelecendo conexão via LLM Chat (Command Center) para organizar a vida pessoal e profissional de acordo com a visão do Personal AI Core.

## User Story

Como **Usuário Principal**, quero **enviar uma mensagem no chat perguntando meus compromissos e tarefas** para **obter um painel consolidado com insights gerados por IA que me ajudem a planejar meu dia de forma otimizada**.

## Acceptance Criteria

- [ ] A IA busca os compromissos e tarefas no Google Calendar e Google Tasks.
- [ ] A aplicação atualiza uma seção "Planejamento" com eventos, tarefas e insights reais.
- [ ] A IA deve analisar os dados (janelas de tempo, sobrecargas) de forma proativa.
- [ ] Interface permite inserir novos dados via Chat.

## Dependencies

- API de Autenticação OAuth do Google.
- Integração Omnibar & LLM Chat.

## Out of Scope

- Qualquer integração ou funcionalidade que não seja estritamente relacionada ao Google Calendar e ao Google Tasks (ex: Microsoft Outlook, envio de e-mails, integrações com outros gerenciadores de tarefas).

## Technical Notes

- OAuth 2.0 flow no backend.
- Persistência criptografada de tokens (PostgreSQL).
- Criação de LangGraph Tools (Python).
